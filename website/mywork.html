My Work

Ant: Mobile Idle Game

Go to project

This is a project I have been working on for quite some time now, this being an early prototype. It will become an idle game built for mobile and released onto the app-store. If you click the link above or the image it will take you to the download page.

In this game the player must slowly build up ant colonies by using food to purchase more queen ants, in turn they are able to birth more and more worker ants. Collect all kinds of ants from the mundane garden ant to the mythical fire ant!

Minecraft Fan Website

Go to project

This is an example of a fan site I created for one of my favorite games, Minecraft. The site includes use of heavy imagery to not overwhelm the visitor with text blocks. I believe the site is very on-brand for the game it's portraying.

The Minecraft fan site makes use of a JavaScript libary to create a very satisfying scroll that makes it feel like four pages in one. Utlizing the brand I wanted it to feel like the user is "digging deeper" just like a player would do in the game itself.

To-Do List Website

Go to project

I created this website to test myself, using impactful imagery I wanted to create a feel of calm and even of awe. This website will use any microphone that you have plugged into your computer and let you add to a to-do list.

Utilizing Bootstrap libaries and JavaScript I managed to create a to-do list that the user can use for any purpose, add items, favorite items, tick items off the list and delete items. The site features a manual way and an audio way to add items to said list.

3D Model Renders

Go to project

These are some assets I created while doing weekly challenges, one being dice and one being egg. I wanted to get better with voxel art and decided this would be the best way to practise.

These rendered versions of the models I created look stunning with the addition of lighting, bloom and matierals. matierals used are as follows; Emmisive for the lighting and glass for the dice body with high roughness to give a plastic feel.
